# Tailwind CSS 꿀팁

Tailwind CSS를 더 효율적으로 사용하기 위한 팁들입니다.

## 1. 그룹 호버

부모 요소에 hover했을 때 자식 요소가 변경되도록 할 수 있습니다.

```html
<div class="group">
  <span class="group-hover:text-blue-500">Hover me!</span>
</div>
```

## 2. 다크 모드

시스템 설정에 따라 자동으로 다크모드를 지원합니다.

```html
<div class="bg-white dark:bg-gray-800">
  자동으로 다크모드 지원!
</div>
```

## 3. 반응형 디자인

모바일 퍼스트로 반응형 디자인을 쉽게 구현할 수 있습니다.

```html
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3">
  반응형 그리드
</div>
```

## 4. 커스텀 색상

tailwind.config.js에서 커스텀 색상을 정의할 수 있습니다.

```javascript
module.exports = {
  theme: {
    extend: {
      colors: {
        brand: '#1a73e8',
      },
    },
  },
};
```

더 많은 팁들은 공식 문서를 참고하세요!
