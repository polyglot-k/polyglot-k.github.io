# 안녕하세요!

이 블로그에 오신 것을 환영합니다. 이것은 첫 번째 블로그 포스트입니다.

## MDX란?

MDX는 마크다운과 JSX를 함께 사용할 수 있게 해주는 형식입니다. 이를 통해:

- 마크다운의 간편한 문법을 사용하면서
- React 컴포넌트를 포함시킬 수 있습니다

## 코드 예시

```javascript
function hello() {
  console.log('Hello, World!');
}
```

## 인용문

> MDX를 사용하면 콘텐츠 작성이 훨씬 즐거워집니다!
