# Next.js 시작하기

Next.js는 React 기반의 풀스택 웹 프레임워크입니다.

## 설치 방법

```bash
npx create-next-app@latest my-app
```

## 주요 특징

1. **파일 기반 라우팅**: pages 또는 app 디렉토리에 파일을 만들면 자동으로 라우트가 생성됩니다.
2. **서버 사이드 렌더링**: 기본적으로 서버에서 페이지를 렌더링합니다.
3. **API Routes**: 백엔드 API를 쉽게 만들 수 있습니다.

## 프로젝트 구조

```
my-app/
├── src/
│   ├── app/
│   │   ├── layout.tsx
│   │   └── page.tsx
│   └── components/
├── public/
└── package.json
```
